# General info
Program for solve and visualize the Dining Philosophers problem.
To run: 
```
$ mkdir build
$ cd build
$ cmake ..
$ make
$ ./dining-philosophers
```
## Algorithm
[Chandi-Misra algorithm](https://www.stolaf.edu/people/rab/pdc/text/dpsolns.htm)
